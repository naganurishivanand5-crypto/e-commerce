# E-commerce Static Site

This repository contains a simple e-commerce static site (HTML/CSS/JS).

Contents:
- index.html, about.html, cart.html, checkout.html, contact.html, login.html, register.html, profile.html, wishlist.html
- `css/`, `js/`, `images/`

To push to GitHub:

1. Create a GitHub repository and copy its remote URL (HTTPS or SSH).
2. Run:

```
cd "c:\my projects\e-commerce"
git remote add origin <REMOTE_URL>
git branch -M main
git push -u origin main
```

If you want me to create the GitHub repo for you, provide a GitHub Personal Access Token with `repo` scope.
